# 🏏 BPL Dream 11 - Ultimate Fantasy Team Builder

![BPL Dream 11 Banner](https://i.ibb.co.com/XG19C3h/banner.png)

**BPL Dream 11** is a modern, high-performance web application designed for cricket enthusiasts to build and manage their fantasy team for the Bangladesh Premier League. Assemble your squad of 6 superstars, manage your budget, and dominate the league!

## 🚀 Live Demo
[View Live Project](https://project-bpl-dream-11.netlify.app/)

---

## ✨ Key Features

### 💰 Dynamic Credit System
Claim free credits with a single click to start building your squad. The app keeps track of your balance in real-time as you purchase world-class players.

### 👥 Professional Player Roster
Browse through a curated list of BPL players. Each player card features:
- High-quality player portraits.
- Nationality and Professional Role (Batsman, Bowler, All-Rounder, etc.).
- Performance Ratings out of 100.
- Detailed Batting & Bowling styles.
- Professional market valuation (Price).

### 🛠️ Team Management Console
- **Toggle Views**: Seamlessly switch between the available market and your selected squad.
- **Validation Engine**: 
    - Prevents selecting the same player twice.
    - Enforces a strict 6-player squad limit.
    - Real-time budget checks before every purchase.
- **Easy Removal**: Changed your mind? Remove players from your squad to reclaim credits and free up spots.

### 🔔 Interactive Feedback
Powered by `react-toastify`, the user receives instant, beautiful notifications for successful selections, budget errors, or duplicate player warnings.

## 🏃‍♂️ How to Run Locally

Follow these steps to get the project up and running on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/bpl-dream-11.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd bpl-dream-11
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
